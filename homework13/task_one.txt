ID: DZ – 13T1
Требование: возможность редактирования чужого репозитория.
Модуль: Fork GitHub.
Имя: Fork репозитория.
Предусловие: 
1.	Открыть Google Chrome. 
2.	В строку поиска ввести https://github.com/. 
3.	Войти в свой репозиторий, ввести логин и пароль.
Шаги:
1.	Открыть необходимый репозиторий пользователя.
2.	Нажать 'Fork'.
3.	Нажать 'Create fork'. 
Ожидаемый результат: копия репозитория пользователя в нашем аккаунте.
Приоритет: Medium.
Постусловие: произвести слияние с репозиторием автора.


ID: DZ – 13T2 
Требование: возможность синхронизироваться с удаленной веткой.
Модуль: Pull Request GitHub.
Имя: закрытие Pull Request из ветки test в ветку master.
Предусловие: 
1.	Открыть Google Chrome. 
2.	В строку поиска ввести https://github.com/.
3.	Войти в свой репозиторий, ввести логин и пароль. 
4.	Создать новую ветку test. 
5.	В ветке создать python файл.
6.	В файле создать новую задачу.
7.	Добавить файл в индекс. 
8.	Создать коммит на новую задачу.
9.	Отравить закоммиченные изменения в удаленный репозиторий.
Шаги:
1.	В нашем репозитории выбрать вкладку 'Pull Request'.
2.	Нажать 'New pull reguest'.
3.	Нажать 'compare: master' и выбрать ветку 'test'.
4.	Нажать 'Create pull reguest' два раза.
5.	Перейти в 'Pull reguests' и выбрать вкладку 'Conversation'. 
6.  Нажать 'Merge pull reguest'. Для закрытия 'Pull reguest' выбрать 'Rebase and Merge'.
6.	Нажать 'confirm rebase and merge'.
Ожидаемый результат: Pull Request закрыт.
Приоритет: Medium.
Постусловие: возвратиться в ветку master.


ID: DZ – 13T3.
Требование: возможность изменения биографии пользователя.
Модуль: Profile GitHub.
Имя: Изменение Bio в профиле пользователя.
Предусловие: 
1.	Открыть Google Chrome. 
2.	В строке поиска ввести https://github.com/.
3.	Войти в свой репозиторий, ввести логин и пароль.
Шаги:
1.	Открыть профиль пользователя.
2.	В списке выбрать имя профиля.
3.	Открыть вкладку 'Overview'.
4.	Под avatar пользователя нажать 'Edit profile'.
5.	Изменить биографию в окне 'Bio'.
6.  Нажать Save.
Ожидаемый результат: появление корректной биографии.
Приоритет: Medium.
Постусловие: закрыть профиль.
